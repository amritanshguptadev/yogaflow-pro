## 📸 About the Project

**YogaFlow Pro** is a professional yoga booking platform with a beautiful UI, smooth animations, interactive components, and Indian localization (₹ pricing, Indian names & phone numbers).

## ⚡ Tech Stack

- **[Astro](https://astro.build)** v5.14.3 — Static site framework
- **[React](https://react.dev)** v19.2.0 — Interactive UI components
- **[Tailwind CSS](https://tailwindcss.com)** v4.1 — Utility-first styling
- **[shadcn/ui](https://ui.shadcn.com)** — Pre-built UI components
- **TypeScript** — Type safety
- **Vite** — Fast build tool

## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/amritanshguptadev/yogaflow-pro.git

# 2. Go into the project folder
cd yogaflow-pro

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev

# 5. Open in browser
# http://localhost:4321
```

## 📂 Project Structure

```
yogaflow-pro/
├── src/
│   ├── pages/          ← Website pages (routes)

│   ├── styles/         ← Global CSS & design tokens
```
