# Security Policy

## Supported Versions

This is an actively maintained project. Security fixes are applied to the latest version only.

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes     |

---

## Reporting a Vulnerability

If you find a security vulnerability in this project, **please do not open a public GitHub issue.**

Instead, email me directly at **amritansh.gupta.dev@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce it
- Potential impact

I'll respond within 48 hours and work on a fix as soon as possible. You'll be credited in the changelog if you'd like.

---

> This project does not handle real payment data or store sensitive user information. It is primarily a frontend demo.
